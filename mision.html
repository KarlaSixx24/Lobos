Datos sobre nosotros:

Se analiza la estrategia de implementar dentro del plantel por medio de las pantallas y anunciar en la página principal y docentes. El sistema de comercio será a través de una página web y un código QR, al inicio de la sesión ingresarán su número de control, para poder realizar la compra o pedido. Se incrementará pago por efectivo, tarjeta y QR. Una vez realizado el pago procederás a ventanilla a recoger el producto. Utilizaremos lenguajes de programación como HTML para las páginas con editor de código Visual Studio, ya que es una herramienta con una interfaz amigable.
